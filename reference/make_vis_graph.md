# Create a visNetwork graph from nodes and edges data frames

Create a visNetwork graph from nodes and edges data frames

## Usage

``` r
make_vis_graph(nodes_df, edges_df, pathway_name)
```

## Arguments

- nodes_df:

  Data frame of nodes.

- edges_df:

  Data frame of edges.

- pathway_name:

  Name of the pathway for the graph title.

## Value

A visNetwork object representing the graph.
