# Convert organism name or abbreviation to KEGG organism code

Convert organism name or abbreviation to KEGG organism code

## Usage

``` r
to_organism_kegg(organism)
```

## Arguments

- organism:

  Organism name or abbreviation (e.g., "human", "hs", "mouse", "mm")

## Value

KEGG organism code (e.g., "hsa" for human, "mmu" for mouse)
