# Convert KEGG IDs with prefixes to IDs without prefixes

Convert KEGG IDs with prefixes to IDs without prefixes

## Usage

``` r
remove_kegg_prefix(kegg_ids)
```

## Arguments

- kegg_ids:

  Character vector of KEGG IDs with prefixes (e.g., "cpd:C00022",
  "mmu:1234", "ko:K00001 ko:K00002")

## Value

List of character vectors with KEGG IDs without prefixes
