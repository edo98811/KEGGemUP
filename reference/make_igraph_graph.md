# Create an igraph graph from nodes and edges data frames

Create an igraph graph from nodes and edges data frames

## Usage

``` r
make_igraph_graph(nodes_df, edges_df, pathway_name)
```

## Arguments

- nodes_df:

  Data frame of nodes.

- edges_df:

  Data frame of edges.

- pathway_name:

  Name of the pathway for the graph title.

## Value

An igraph object representing the graph.
