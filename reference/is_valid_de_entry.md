# Check the structure and validity of each entry in de_results

Check the structure and validity of each entry in de_results

## Usage

``` r
is_valid_de_entry(de_entry, name)
```

## Arguments

- de_entry:

  An entry from the de_results list

- name:

  The name of the entry in the de_results list (for error messages)

## Value

Invisible TRUE if all checks pass, otherwise stops with an error
