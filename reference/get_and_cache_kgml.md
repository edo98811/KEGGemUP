# Download and cache KEGG KGML files.

Download and cache KEGG KGML files.

## Usage

``` r
get_and_cache_kgml(pathway_id, bfc)
```

## Arguments

- pathway_id:

  KEGG pathway ID (e.g., "hsa04110").

- bfc:

  BiocFileCache object for caching KEGG KGML files.

## Value

Path to the cached KGML file.
